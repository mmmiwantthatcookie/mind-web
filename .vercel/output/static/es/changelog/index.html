MIND v1.0 — Lanzamiento Público Inicial

El primer lanzamiento público de MIND, el pipeline de Detección de Inconsistencias Multilingüe, acompañando el artículo presentado en EMNLP 2025.

Destacado

  • Backend multi-LLM con soporte para GPT-4, Claude y modelos de código abierto
  • Modelado de temas polilingual usando embeddings multilingüe de última generación
  • Sistema de recuperación híbrido combinando búsqueda densa y dispersa
  • Panel web interactivo para revisar las discrepancias detectadas
  • Despliegue basado en Docker para fácil configuración y escalado
  • Dataset en HuggingFace publicado para benchmarking
