## v0.2.1 — Correcciones de bugs y búsqueda avanzada

Correcciones en la interfaz web, validación mejorada en formularios, y nuevo soporte para carga de datasets. Opciones de búsqueda avanzada añadidas pero deshabilitadas por defecto. Mejoras para retrievals más específicos llegarán pronto a MIND.

### Correcciones

#### Interfaz web y Formularios
- Formulario de registro mantiene email/usuario si falla validación (sin reescribir)
- Eliminar datos (Zona de Peligro), bugs solucionados, ahora elimina correctamente toda la data.
- Zona de Peligro rediseñada para mayor claridad y simplicidad de uso.

#### Backend y Detección
- Resultados de detección ordenados por pares (antes inconsistente)
- Carga de prompts soporta rutas alternativas para despliegues más flexibles

### Nuevas Capacidades

#### Gestión de Datasets
- Carga de datasets disponible en la vista `Datasets`, se mantiene la opción de carga desde el perfil del usuario para la retrocompatibilidad.
#### Búsqueda Avanzada (Opcional)
Nuevos flags de configuración (deshabilitados por defecto):
- `use_cross_encoder_rerank` — reranking con BAAI/bge-reranker-v2-m3
- `use_bidirectional_retrieval` — combina puntuación hacia adelante + inversa (`bidirectional_alpha: 0.6`)

Habilitalas en `config.yaml` para probarlo; no recomendado aún para producción.

### Notas

- Sin cambios de ruptura; retrocompatible
- Reemplazo directo de v0.2.0-alpha
